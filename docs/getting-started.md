# Getting Started

## 1. Install dependencies

```bash
cd superta
npm install
```

## 2. Build the scaffold

```bash
npm run build
```

## 3. Run tests

```bash
npm test
```

## 4. Copy the example config

```bash
cp config.example.json local.config.json
```

Then edit it for your local environment.

## 5. Read the key docs

Start with:
- `docs/architecture.md`
- `docs/config.md`
- `docs/demo-workflow.md`
- `docs/milestones.md`

## 6. Pick a development path

Suggested next focus areas:
- provider integration
- Gmail integration
- proposal application
- packaging / install flow
